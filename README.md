LIBRARY SYSTEM (WPF C#)

ОПИСАНИЕ ПРОЕКТА:
Это WPF приложение на языке C#.
Система библиотеки для управления книгами и журналами с графическим интерфейсом.

ФУНКЦИОНАЛ:
- Добавление книги
- Добавление журнала
- Просмотр всех элементов
- Редактирование записей
- Удаление записей
- Просмотр статистики

АРХИТЕКТУРА ПРОЕКТА:

MODELS:
- LibraryItem (базовый класс)
- Book (книга)
- Magazine (журнал)

REPOSITORY:
- Хранение данных в памяти (List)
- Add
- Remove
- GetAll

SERVICE:
- Бизнес-логика приложения
- AddBook
- AddMagazine
- Update
- Remove
- GetAll
- GetStats

USER INTERFACE (MainWindow):
- Поля ввода:
  * Название
  * Год издания
  * Автор (для книги) / Выпуск (для журнала)

- Кнопки:
  * Добавить
  * Редактировать
  * Удалить
  * Статистика

- Таблица данных:
  DataGrid отображает:
  ID | Тип | Название | Год | Автор / Выпуск

ОСОБЕННОСТИ UI:
- Используется DataGrid для отображения данных
- Обновление данных через Refresh
- Работа через Service слой

МЕНЮ РАБОТЫ:
1. Ввод данных пользователя
2. Добавление записи в библиотеку
3. Редактирование выбранной записи
4. Удаление записи
5. Просмотр статистики

ТЕХНОЛОГИИ:
C#
WPF (Windows Presentation Foundation)
XAML
ООП (классы, наследование, полиморфизм)
Repository pattern
Service layer
